## Discord Personal Theme
<p>Code to a theme I edited for my personal likeing.</p>



## Overview
<p>Gives a red asthetic to Discord's usual look</p>



## How do I use the theme?
<li>First, you must download <a href="https://vencord.dev/download/">Vencord</a>.</li>
<li>After downloading Vencord, go to themes and click "Edit QuickCSS"</li>
<li>Paste this into the CSS box</li>

```css
@import url(https://raw.githubusercontent.com/Entertalned/PersonalTheme/refs/heads/main/MyTheme.css)
```
